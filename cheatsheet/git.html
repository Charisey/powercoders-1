Git Cheatsheet

Git is a Version Control System. Git has 3 location.

  • Working Directory
  • Index
  • Repository

Git commands copy files between these location.

Git Workflow

              Working directory
                    ↓↑
                git diff ...
                    ↓
                git add ...
                    ↓
                  Index
                    ↓
                git commit ...
                    ↓
                Repository

Git Init

  • Tell Git where the root of the Repository is.
  • Example: %.....poco>git init
  • Now everything in the poco folder can be managed by git

Git Status

  • Lists all new or modified files to be commited
  • %git status

Git Add

  • Git add will add the files to index.
  • %git add file-name
  • %git add file1.ext file2.ext file3.ext to add multiple files

Git Commit

  • git commit will move the files in the index to repository.
  • %git commit -m "message"

Git diff

  • Displays the difference between all the files in Working Directory and the file in index or repository
  • %git diff
  • git diff --cached (or --staged) displays the differnece between file in index and repository

Git log

  • Display commit logs.
  • %git log
